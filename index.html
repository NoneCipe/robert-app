01

СКАЧАЙТЕ ПРИЛОЖЕНИЕ

Нажмите кнопку выше и скачайте ROBERT.exe. Установка не требуется — просто запустите файл.

02

ПОЛУЧИТЕ БЕСПЛАТНЫЙ API КЛЮЧ

Зарегистрируйтесь на console.groq.com → API Keys → Create API Key. Это бесплатно и бессрочно.

03

ВВЕДИТЕ КЛЮЧ В НАСТРОЙКАХ

Запустите ROBERT.exe → нажмите ⚙ → вставьте ключ → сохраните. Готово!

Quick
start

Процесс запуска не менял: пользователь скачивает файл, получает ключ и через пару минут уже может пользоваться помощником.

01

Скачайте приложение

Нажмите кнопку выше и скачайте ROBERT.exe. Установка не требуется.

02

Получите API-ключ

Зарегистрируйтесь на console.groq.com и создайте API Key. Это бесплатно.

03

Вставьте ключ в настройки

Запустите ROBERT, откройте настройки, вставьте ключ и сохраните. После этого помощник готов к работе.
